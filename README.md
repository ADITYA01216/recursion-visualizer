# Recursion Visualizer

An interactive web-based tool to visualize recursion algorithms and understand function call stacks.

## Features
- Step-by-step recursion visualization
- Stack call representation
- Code highlighting
- Multiple recursion problems

## Tech Stack
- React
- Vite
- JavaScript
- CSS
